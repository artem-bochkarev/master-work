Решение задачи об умном муравье с использованием технологии OpenCL

Автор: Бочкарев Артем, artem.bochkarev@gmail.com

Для успешной сборки проекта необходимо:

1. Библиотека Boost и переменная среды с именем "BOOST"
2. Библиотека OpenCL и переменная среды с именем "OpenCL"
3. Библиотека wxWidgets или Qt( для графической части. Если необходимы только измерения скорости выполнения, можно отключить эти проекты ).
